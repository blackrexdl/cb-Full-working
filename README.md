# Cylinder Booking System

## About Project

This Cylinder Booking System is designed to simplify the process of booking gas cylinders online. Built with PHP, HTML, and Vanilla CSS, it offers a user-friendly interface for customers and an admin panel for managing bookings and users efficiently.

A **Cylinder Booking System** built using **PHP, HTML, and Vanilla CSS**.  
This project allows users to **book cylinders easily** with a clean and responsive interface.
